Way-2 Credenciales almacenadas en el sistema.

Si enumeramos el sistema con PowerUp.ps1 obtenemos una ruta con unas credenciales almacenadas en un archivo xml

GPP Passwords (preferencias de directivas de grupo) Cada vez que se crea una nueva preferencia de grupo, se crea un archivo xml en el recurso compartido SYSVOL con esos datos de configuracion, incluidas las contraseñas asociadas co el GPP. Por seguridad, Microsoft AES encripta la contraseña antes de almacenarla como cpassword, ¡pero luego Microsoft publico la clave en MSDN!. Se pueden decifrar con la tool gpp-decrypt
